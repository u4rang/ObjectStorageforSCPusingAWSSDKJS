# ObjectStorageforSCPusingAWSSDKJS
Guide Object Storage for SCP using AWS SDK Javascript

### 환경설정

#### npm init
```sh
npm init -y
```

#### aws-sdk 설치
```sh
npm install aws-sdk
```

#### endpoint 주소
 - https://obj1.kr-east-1.samsungsdscloud.com:8443

### 실행
```sh
npm index.js
```
